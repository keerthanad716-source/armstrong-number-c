# Armstrong Number Check in C

This C program checks whether a given number is an Armstrong number or not.

## What is an Armstrong Number?
An Armstrong number is a number that is equal to the sum of its digits raised to the power of the number of digits.

Example:
153 = 1³ + 5³ + 3³ = 153

## How to Run
1. Compile the program  
   gcc armstrong.c -o armstrong -lm
2. Run the program  
   ./armstrong
